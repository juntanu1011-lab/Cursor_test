# 日報管理システム

日報データを効率的に管理・分析するためのシステムです。

## 機能

- 📋 **日報の自動フォーマット化**: 生データをテンプレート形式に変換
- 📊 **タスク分析ダッシュボード**: インタラクティブなデータ可視化
- 📅 **週報の自動生成**: 複数日の日報から週報を自動生成
- 📝 **Marp形式のプレゼンテーション**: 業務報告をスライド形式で作成

## プロジェクト構造

```
.
├── input/          # 入力データ
│   ├── 日報テンプレ.md
│   └── 20251229.md
├── output/         # 出力データ
│   ├── 20251223.md 〜 20251229.md  # フォーマット化された日報
│   ├── 週報_20251223-20251229.md   # 週報サマリー
│   ├── task_analysis.html           # タスク分析ダッシュボード
│   ├── プロジェクト説明資料.md
│   └── 今週の業務報告_20251223-20251229.md
└── index.html      # メインページ
```

## 使用方法

### タスク分析ダッシュボード

1. `output/task_analysis.html` を開く
2. 日報ファイル（Markdown形式）をアップロード
3. 自動的にタスクが分析され、チャートで可視化されます

### 日報の表示

各日の日報は `output/` フォルダ内のMarkdownファイルから確認できます。

## デプロイ

このプロジェクトはGitHub Pagesでデプロイできます。

### GitHub Pagesでのデプロイ方法

1. GitHubリポジトリの **Settings** を開く
2. 左メニューから **Pages** を選択
3. **Source** で **Deploy from a branch** を選択
4. **Branch** で **main** を選択し、**/ (root)** を選択
5. **Save** をクリック

数分後、`https://[ユーザー名].github.io/Cursor_test/` でアクセスできます。
